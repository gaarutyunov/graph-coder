# graph-coder

GraphCoder: Transformer Pre-Trained on Code Represented as Graph

## Prerequisites

1. Install Python [3.9](https://www.python.org/downloads/release/python-390/)
2. Install [Poetry](https://python-poetry.org/docs/#installation)
3. Install [Poethepoet](https://github.com/nat-n/poethepoet/#installation)
4. Install dependencies: `poetry install`
